Introduction

JNSViewer is an online tool for visualization of the RNA or DNA Secondary Structure from a DBN or CT format file, or a nucleotide sequence. The purpose of this manual is to provide an extensive instruction of how to use JNSViewer and its features.


Instructions

1. Import data

  • Upload the predicted structure file and use our server to visualize the structure

    1. Go to other websites to get the predicted secondary structure file and upload it to our server. The supported formats are SSDJ, DBN and CT. The maximum file size allowed is 1 MB.

    2. [optional] GFF information.

    3. Click the "Submit" button.

  • Once your sequence has been processed, you will receive the following page. Please remember to save the web link and access code if you need to recover your project for later.

2. Structure viewer

  • Once data has been processed, click "Highly interactive secondary structure viewer" in the new page.

  • A new window will open with the sequence, GFF annotation, DBN structure, and image of your sequence. You can use "Rulers" to show the sequence in a formatted way. You can also change the color mode to either "GFF" or "Nucleotide".

  • The color key and zoom controls for SVG graph are located on the top of the structure graph.There are hotkeys available for the zoom controls ("+" and "-").

  • Advanced functions
    • Motif search: Search of the nucleotide sequence can be done using the search box within the "Nucleotide Information". Regular expression can be used to search for the motif. A minimum of 3 characters is required. If multiple results are returned, you can use the "Next" and "Previous" buttons to navigate the results. The result motif will be highlighted in the nucleotide sequence, structure DBN file, and structure graph.

    • Show 2D structure of selected bases: When the structure graph is complex, you can select sections of the large structure to then view the sub-structure. To select the sub-structure, you can (1) use mouse to select (drag) bases in the nucleotide sequence; (2) hold "Shift" and click bases in the structure DBN file; or (3) hold "Shift" and click bases in the structure graph.
      Notice: You must select the complete paired bases to show the sub-structure.
      Click "Show 2D structure of selected bases", to view your sub-structure.

3. Other result links

  • On the result page, you can find other useful links. Data can be downloaded by clicking on the corresponding link.

4. Retrieve project

To retrieve the results of your previous project, click the "Retrieve" tab on the top navigation bar. Enter your User ID and Access code. The data will be stored and available for one week.
